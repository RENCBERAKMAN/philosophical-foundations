# philosophical-foundations

> *"Sorgulanmamış bir hayat yaşanmaya değmez."* — Sokrates

Felsefi gelişimimi takip ettiğim kişisel bilgi arşivi. Ders kitabı değil; kavram kavram, argüman argüman inşa edilen bir düşünce altyapısı.

---

## Neden?

Teknik beceri, kavramsal derinlik olmadan kırılgandır. Sistemlerin *nasıl* çalıştığını anlamak, *neden* öyle yapılandırıldıklarını anlamakla aynı şey değil.

Bu repo o ikinci soruya cevap aramak için var.

---

